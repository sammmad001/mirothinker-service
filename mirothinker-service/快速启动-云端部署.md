## 下一步：上传到云端

### 方法 1: 打包上传 (推荐)

**在本地 Mac 上执行：**
```bash
cd /Users/sam/Desktop/mirothinker-service
./package-for-deploy.sh
```

这会生成 `mirothinker-service-YYYYMMDD_HHMMSS.tar.gz` 在桌面。

**上传到服务器：**
```bash
scp /Users/sam/Desktop/mirothinker-service-*.tar.gz user@your-server-ip:~/
ssh user@your-server-ip
tar xzf mirothinker-service-*.tar.gz
cd mirothinker-service
```

### 方法 2: Git 推送

```bash
cd /Users/sam/Desktop/mirothinker-service
git init
git add .
git commit -m "Ready for cloud deployment"
git remote add origin <your-repo-url>
git push -u origin main
```

**在服务器上：**
```bash
git clone <your-repo-url> ~/mirothinker-service
cd ~/mirothinker-service
```

### 方法 3: Rsync 直接同步

```bash
rsync -avz --progress \
  --exclude='.env' \
  --exclude='traces' \
  --exclude='logs' \
  --exclude='cache' \
  --exclude='.git' \
  /Users/sam/Desktop/mirothinker-service/ \
  user@your-server-ip:~/mirothinker-service/
```

## 云端服务器部署

### 1. 服务器要求
- Ubuntu 20.04/22.04 或 CentOS 8+
- 最低 2核CPU, 2GB内存
- Docker 和 Docker Compose 已安装

### 2. 配置环境变量
```bash
cd ~/mirothinker-service
cp .env.example .env
nano .env
```

**必须配置：**
```env
DASHSCOPE_API_KEY=sk-xxx  # 你的百炼API密钥
```

### 3. 一键部署
```bash
chmod +x cloud-deploy.sh
./cloud-deploy.sh
```

### 4. 验证部署
```bash
# 检查服务状态
docker-compose ps

# 测试健康检查
curl http://localhost/api/health

# 查看日志
docker-compose logs -f app
```

### 5. 访问服务
- **前端**: http://your-server-ip
- **API**: http://your-server-ip/api/health

