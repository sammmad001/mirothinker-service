## 执行研究测试

### 测试主题

**查询**: "分析每股光模块、CPO和存储产业、投资标的和投资策略"

### 发起研究

```bash
curl -X POST http://127.0.0.1:8000/api/research \
  -H "Content-Type: application/json" \
  -d '{
    "query": "分析每股光模块、CPO和存储产业、投资标的和投资策略",
    "max_turns": 150,
    "context_keep": 5,
    "model": "qwen-plus",
    "temperature": 0.7
  }'
```

**预期返回**:
```json
{
  "task_id": "abc12345",
  "status": "running"
}
```

### 查询进度

```bash
curl http://127.0.0.1:8000/api/research/abc12345 | python3 -m json.tool
```

**运行中状态**:
```json
{
  "task_id": "abc12345",
  "status": "running",
  "turn_count": 25,
  "elapsed_time": 45.3
}
```

### 获取结果

研究完成后 (通常 3-8 分钟):

```bash
curl http://127.0.0.1:8000/api/research/abc12345 | python3 -m json.tool
```

**完整结果包含**:
- `result`: 研究报告 (Markdown 格式)
- `domain`: 识别的领域 (应为 "finance")
- `quality_report`: 质量评分
- `metadata`: 来源列表和可信度评分

## 预期研究内容

### 光模块产业

研究将涵盖:
- 全球光模块市场规模和增长趋势
- 主要技术路线 (100G/400G/800G)
- 核心厂商分析 (中际旭创、光迅科技、新易盛等)
- 每股关键指标 (EPS, PE, PB)
- 投资策略建议

### CPO (共封装光学)

研究将涵盖:
- CPO 技术原理和优势
- 产业链分析 (设计、制造、封装)
- 主要参与企业 (博创科技、天孚通信等)
- 市场预测和竞争格局
- 投资机会和风险

### 存储产业

研究将涵盖:
- 存储芯片市场 (DRAM, NAND, NOR)
- 行业周期分析
- 国内替代机会 (兆易创新、北京君正等)
- 价格趋势和供需关系
- 长期投资逻辑
