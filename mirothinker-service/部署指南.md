## 五、部署步骤

### 5.1 服务器要求

| 要求 | 最低配置 | 推荐配置 |
|------|----------|----------|
| CPU | 2 核 | 4 核 |
| 内存 | 4 GB | 8 GB |
| 带宽 | 5 Mbps | 10+ Mbps |
| 存储 | 20 GB | 40 GB |
| 系统 | Ubuntu 20.04+ | Ubuntu 22.04 LTS |

### 5.2 部署流程

#### 步骤 1: 上传代码到服务器

```bash
# 在本地打包
cd /Users/sam/Desktop
tar -czf mirothinker-service.tar.gz mirothinker-service/

# 上传到服务器
scp mirothinker-service.tar.gz root@your-server-ip:/opt/

# 在服务器上解压
ssh root@your-server-ip
cd /opt
tar -xzf mirothinker-service.tar.gz
mv mirothinker-service /opt/mirothinker
```

#### 步骤 2: 配置环境变量

```bash
cd /opt/mirothinker
cp .env.example .env
nano .env

# 填入以下 API Key:
DASHSCOPE_API_KEY=sk-xxx          # 百炼 API Key
SERPER_API_KEY=xxx                 # Serper API Key
JINA_API_KEY=xxx                   # Jina API Key
```

#### 步骤 3: 安装依赖

```bash
cd /opt/mirothinker/backend
pip3 install -r requirements.txt
```

#### 步骤 4: 启动服务

```bash
# 测试启动
cd /opt/mirothinker/backend
python3 main.py

# 访问 http://your-server-ip:8000
```

#### 步骤 5: 生产部署 (systemd)

```bash
# 创建 systemd 服务
sudo tee /etc/systemd/system/mirothinker.service << 'EOF'
[Unit]
Description=MiroThinker Online Service
After=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/opt/mirothinker/backend
EnvironmentFile=/opt/mirothinker/.env
ExecStart=/usr/bin/python3 -m uvicorn main:app \
    --host 127.0.0.1 \
    --port 8000 \
    --workers 2
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
EOF

# 启动服务
sudo systemctl daemon-reload
sudo systemctl enable mirothinker
sudo systemctl start mirothinker
sudo systemctl status mirothinker
```

#### 步骤 6: 配置 Nginx

```bash
# 复制 Nginx 配置
sudo cp /opt/mirothinker/nginx.conf /etc/nginx/sites-available/mirothinker
sudo ln -s /etc/nginx/sites-available/mirothinker /etc/nginx/sites-enabled/

# 修改配置中的域名
sudo nano /etc/nginx/sites-available/mirothinker

# 测试并重启
sudo nginx -t
sudo systemctl restart nginx
```

#### 步骤 7: SSL 证书 (可选)

```bash
# 安装 Certbot
sudo apt install certbot python3-certbot-nginx

# 获取证书
sudo certbot --nginx -d your-domain.com
```

## 六、API 获取指南

### 6.1 百炼 DashScope API Key

1. 访问 https://bailian.console.aliyun.com/
2. 登录阿里云账号
3. 进入「API-KEY 管理」页面
4. 创建新的 API Key
5. 复制 Key 到 `.env` 文件的 `DASHSCOPE_API_KEY`

### 6.2 Serper API Key

1. 访问 https://serper.dev/
2. 注册账号
3. 进入 Dashboard → API Keys
4. 复制 Key 到 `.env` 文件的 `SERPER_API_KEY`
5. 免费额度: $100 (约 5,000 次搜索)

### 6.3 Jina API Key

1. 访问 https://jina.ai/
2. 注册账号
3. 进入 Dashboard 获取 API Key
4. 复制 Key 到 `.env` 文件的 `JINA_API_KEY`
5. Reader API 目前免费

## 八、安全注意事项

1. **API Key 安全**: `.env` 文件不要提交到 Git
2. **HTTPS**: 生产环境必须使用 HTTPS
3. **限流**: 建议在 Nginx 中配置请求限流
4. **输入验证**: 后端已实现基本验证，建议增加内容安全审查
5. **日志脱敏**: 不要在日志中记录 API Key
