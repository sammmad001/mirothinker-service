# ContentAI - AI Social Media Content Generator

🚀 AI-powered content generation for Southeast Asia & Middle East markets

## Features

- 🎯 Generate content for Instagram, Facebook, Twitter, TikTok
- 🌍 Support for English, Arabic, Malay, Indonesian
- 💰 Simple pricing: Free, $9.99/mo Pro, $29/mo Enterprise

## Quick Start

Open `dist/index.html` in your browser.

## Deploy

### Netlify
Drag the `dist` folder to [app.netlify.com/drop](https://app.netlify.com/drop)

### Vercel
```bash
npx vercel --prod
```

### GitHub Pages
Push to main branch, enable Pages in settings.

## Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- No dependencies required
