# ContentAI - 立即执行清单

## 第1步：部署（5分钟）

### 方式A：Netlify（推荐，免费，最快）
1. 打开 https://app.netlify.com/drop
2. 直接拖拽 `ai-content-generator/dist` 文件夹
3. 获得一个免费URL，如 `random-name-123.netlify.app`
4. 复制URL用于推广

### 方式B：Vercel
```bash
npm i -g vercel
cd ai-content-generator
vercel --prod
```

## 第2步：发布到Product Hunt（10分钟）

1. 访问 https://producthunt.com/post
2. 用GitHub/Google登录
3. 选择 "Maker" 模式
4. 填写：
   - Tagline: "Generate social media content in Arabic, Malay, Indonesian & English - in seconds."
   - Description: 使用 `copy-templates.md` 里的 Product Hunt 部分
   - Website: 你的Netlify URL
   - Categories: AI, Productivity
5. 发布！
